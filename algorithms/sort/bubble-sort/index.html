Bubble Sort

Also known as Sinking sort. Works by repeatedly swapping adjacent elements that are out of order.

func BubbleSort(a []int) {
  if len(a) == 0 {
    return
  }

  // â–§ processed element
  // â–¡ ignored element
  // â–£ cursored element (where i is at)
  for i := 0; i < len(a)-1; i++ { // â–§â–§â–§â–§â–§â–¡ all but last
    for j := i + 1; j < len(a); j++ { // â–¡â–£â–§â–§â–§â–§ everything after i
      if a[i] > a[j] {
        a[i], a[j] = a[j], a[i]
      }
    }
  }
}

func TestFuzzy(t *testing.T) {
  var treat, ctrl []int
  rand := rand2.New(rand2.NewSource(100))

  for i := 0; i < 100; i++ {
    value := rand.Intn(50)
    treat = append(treat, value)
    ctrl = append(ctrl, value)
  }

  slices.Sort(ctrl)
  BubbleSort(treat)

  result := fmt.Sprintf("%v", treat)
  expected := fmt.Sprintf("%v", ctrl)

  if result != expected {
    t.Fatalf("expected %s, but got %s", expected, result)
  }
}

Complexity

The outermost loop has size \(n-1\). The inner loop performs \(n-1\) iterations in first loop, \(n-2\) in second loop, \(n-3\) in third loop and so on.

\[ \begin{alignat}{1} (n-1) + (n-2) + (n-3) + \dots + 1 &= \text{sum of first }n-1\text{ natural numbers} \\ &= (n-1) \cdot \dfrac{1 + (n-1)}{2} \\ &= O(n^2) \end{alignat} \]
Scenario Comparisons Swaps
Worst \(O(n^2)\) \(O(n^2)\)
Average \(O(n^2)\) \(O(n^2)\)
Best \(O(n^2)\) \(O(1)\)
