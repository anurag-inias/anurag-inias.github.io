## Exponent

- $a^0 = 1$
- $a^1 = a$
- $a^{-m} = \dfrac{1}{a^m}$
- $a^m \times a^n = a^{m+n}$
- $a^m \div a^n = \dfrac{a^m}{a^n} = a^{m-n}$
- $(a^m)^n = a^{mn}$
- $a^mb^m = (ab)^m$
- $a^{\frac{1}{m}} = \sqrt[m]a$

## Logarithmic

- $\log_b{m} = n \text{ if } b^n = m \text{ where } b \not = \lbrace0, 1\rbrace$
- $\log_b{1} = 0$
- $\log_b{b} = 1$
- $\log_b{0} = \text{undefined}$
- $\log_b{mn} = \log_b{m} + \log_b{n}$
- $\log{\dfrac{m}{n}} = \log{m} - \log{n}$
- $\log_b{m^n} = n\log_b{m}$
- $\log_c{m} = \dfrac{\log_b{m}}{\log_b{c}}$
- $m^{\log_b{n}} = n^{\log_b{m}}$
