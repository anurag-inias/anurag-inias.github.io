Graph coloring

Description

Graph coloring is the assignment of labels (traditionally called "colors") to vertices in a graph. The key constraint in this assignment is that no two adjacent vertices can have the same color.

Our goal is to find the fewest number of color to label all vertices.

Tip

Every planar graph can be 4-colored.

Graph coloring is a NP-complete problem. We can use greedy approach, but understand that it will not always give an optimal solution.

data class Degree(val node: Int, val degree: Int)

// Super simple implementation of an unweighted undirected graph.
// Nodes are expected to be contiguously numbered, starting from 1.
class Graph {
  val neighbours = mutableMapOf<Int, MutableList<Int>>()

  fun connect(first: Int, second: Int): Graph {
    neighbours.putIfAbsent(first, mutableListOf())
    neighbours.putIfAbsent(second, mutableListOf())

    neighbours[first]!! += second
    neighbours[second]!! += first

    return this
  }
}

enum class Color(val key: Int) {
  BLUE(1),
  GREEN(2),
  RED(3),
  YELLOW(4),
  PINK(5),
  WHITE(6),
  PURPLE(7),
}

// Map a color number to a human readable color name.
val Int.color: String
  get() {
    return Color.entries.filter { it.key == this }.map { it.name }.firstOrNull() ?: "$this"
  }

fun Graph.printColors(colors: MutableMap<Int, Int>) {
  println(neighbours.keys.sorted().map { colors[it]!!.color })
}

Solution

Pseudocode
  • Create a max-heap of all vertices where the key is degree of each vertex.
  • Poll vertices from max-heap one by one and assign a color not used by any of its neighbour.

fun greedyColoring(graph: Graph) {
  // Max heap.
  val degrees = PriorityQueue<Degree>(graph.neighbours.size) { a, b -> b.degree.compareTo(a.degree) }
  for (node in graph.neighbours.keys) {
    degrees.offer(Degree(node, graph.neighbours[node]?.size ?: 0))
  }

  val color = mutableMapOf<Int, Int>()
  while (degrees.isNotEmpty()) {
    val (node, _) = degrees.poll()
    val neighbourColors = (graph.neighbours[node] ?: listOf()).filter { it in color }.map { color[it] }

    for (c in 1..<100) {
      if (c in neighbourColors) continue
      color[node] = c
      break
    }
  }
  graph.printColors(color)
}
