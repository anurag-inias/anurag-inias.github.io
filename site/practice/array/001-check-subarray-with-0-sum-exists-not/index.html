Check if a subarray with 0 sum exists or not

Description

For an integer array, find out if any subarray (i.e. contiguous slice) exists with sum zero.

Example

Array = 3, 4, -7, 3, 1, 3, 1, -4, -2, -2

Zero sum slices:
{ 3, 4, -7 }
{ 4, -7, 3 }
{ -7, 3, 1, 3 }
{ 3, 1, -4 }
{ 3, 1, 3, 1, -4, -2, -2 }
{ 3, 4, -7, 3, 1, 3, 1, -4, -2, -2 }

Solution

Expand

Maintain running sum left to right and keep saving that in a set. If a sum is seen again, that means the subarray with elements ending at current element sum up to be zero.

fun subarraysOfZeroSum(array: IntArray) {
  // [a, b, ...]
  // in this case sumSeen[a] = [0], and sumSeen[0] = [-1]
  val sumsSeen = mutableMapOf(0 to mutableListOf(-1))

  var sum = 0
  for ((i, n) in array.withIndex()) {
    sum += n

    if (sum in sumsSeen) {
      for (start in sumsSeen[sum]!!) {
        val slice = array.sliceArray((start + 1)..i)
        println("Found subarray %s with zero sum".format(slice.contentToString()))
      }
    }

    sumsSeen.putIfAbsent(sum, mutableListOf())
    sumsSeen[sum]!!.add(i)
  }
}
