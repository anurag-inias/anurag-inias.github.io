    Longest substring without repeating characters

    Problem

    Given a string, find the longest substring in it without repeating characters.

    Examples

    Input Output
    abcabcbb 3 (abc)
    bbbb 1 (b)
    pwwkew 3 (wke)
    Solution

    Since we are looking for contiguous section (i.e. substring), use a running window.

    The idea is to keep adding characters to this running window and thus keep increasing the size of candidate substring as long as the characters encountered are unique.

    As soon as we hit a duplicate, we need to reset the running window. Say we encounter a character first time at index 2 and then again at 5, then index 2 is spoiled.

    Implementation 
    int lengthOfLongestSubstring(String s) {
  if (s == null) return 0;
  if (s.length() < 2) return s.length();

  Map<Character, Integer> seenAt = new HashMap<>();
  int maxLength = 0;
  for (int i = 0; i < s.length(); i++) {
    char c = s.charAt(i);

    if (seenAt.containsKey(c)) {
      // found a duplicate character.
      int priorIndex = seenAt.get(c);
      // reset running window.
      seenAt.entrySet().removeIf(entry -> entry.getValue() <= priorIndex);
      seenAt.put(c, i);
    } else {
      // all good. keep appending to candidate substring.
      seenAt.put(c, i);
      maxLength = Math.max(maxLength, seenAt.size());
    }
  }

  return maxLength;
}
