    Demo

    Heapify

    Takes an array of length \(N\) and calls sink on indices \(\Big[\lfloor \frac{N-1}{2} \rfloor, ..., 0\Big]\) in order. Has the amortized time complexity of \(O(n)\).

    Push

    Takes the heap and pushes the element at the end of it. This is followed by swim operation on newly added element to bubble it up to its right place. Has the time complexity of \(O(log \ n)\).

    Pop

    Takes the heap and its root. The root gets replaced with the last element of the heap, followed by sink operation on the new root to restore the min-heap property. Has the time complexity of \(O(log \ n)\).
